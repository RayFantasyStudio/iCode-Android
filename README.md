## 引入
### 在 IntelliJ IDEA 或 Android Studio 中
clone项目到本地。
在项目根目录创建`bugly.properties`，写入以下内容（bugly官方网址<http://bugly.qq.com/>）
```
appId=申请的bugly的appId
appKey=申请的bugly的appKey
```
在IDE中使用File->Open打开即可
